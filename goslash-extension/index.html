How It Works

1

Type "go" + Tab

Activate GoSlash from your address bar

2

Enter shortcut

Type a keyword like yt, gh, or gm

3

Press Enter

You're there. That's it.

Examples

go yt → youtube.com
go gm → mail.google.com
go gh/microsoft → github.com/microsoft
go g how to code → Google search for "how to code"
go / → Open GoSlash settings

Features

Custom Shortcuts

Create your own shortcuts for any website. Add as many as you need.

Usage Statistics

Track which shortcuts you use most. Sort by usage to find your favorites.

Dynamic Templates

Search & Filter

Quickly find any shortcut with the built-in search on the settings page.

Import & Export
