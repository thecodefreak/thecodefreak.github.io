GoSlash Install for Chrome

Privacy Policy

Last updated: February 7, 2026

Overview

GoSlash is a browser extension that allows users to create and use keyboard shortcuts for quick navigation to websites. We are committed to protecting your privacy.

Data Collection

GoSlash does not collect, transmit, or share any personal data.

Data Stored Locally

GoSlash stores the following data locally on your device using Chrome's sync storage:

  • Shortcuts: The keywords, URLs, and titles you create for quick navigation.
  • Usage Statistics: A count of how many times each shortcut has been used (stored locally, never transmitted).

Chrome Sync

If you are signed into Chrome and have sync enabled, your shortcuts will sync across your devices using Chrome's built-in sync functionality. This is handled entirely by Chrome and Google's infrastructure. GoSlash does not operate any servers or have access to your synced data.

Data Sharing

GoSlash does not:

  • Collect any personal information
  • Track your browsing history
  • Send any data to external servers
  • Use any analytics or tracking services
  • Display advertisements

Permissions

GoSlash requires the following permission:

  • storage: To save your shortcuts and usage statistics locally and sync them across your devices via Chrome sync.

Third-Party Services

GoSlash does not integrate with or send data to any third-party services.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date above.

Open Source

GoSlash is open source. You can review the complete source code on GitHub to verify our privacy practices.

Contact

If you have any questions about this privacy policy, please open an issue on our GitHub repository.
